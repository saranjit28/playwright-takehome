# Playwright Take Home Assignment

## Tech Stack

- Playwright
- TypeScript
- Page Object Model

## Node Version

Node.js LTS

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```

## Run in Headed Mode

```bash
npx playwright test tests/todo.spec.ts --headed
```

## Framework Design

- Used Page Object Model for maintainability
- Used Playwright Test Runner
- Used stable locators and assertions
- Kept framework lightweight and readable
